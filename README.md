# Todo App 📝

A simple and minimal Todo application built using HTML, CSS, and JavaScript.  
It allows users to add and delete tasks easily.

---

## 🚀 Features

- Add new tasks
- Delete tasks
- Simple and clean UI
- Lightweight and fast

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)

---

## 📂 Folder Structure

│

├── index.html

├── style.css

├── app.js

└── README.md

## How to Run
Open `index.html` in your browser.
